# Weather Icons

Coloca aqui los SVG del clima.

Archivos esperados:

- `sunny.svg`
- `partly-cloudy.svg`
- `cloudy.svg`
- `rain.svg`
- `storm.svg`
- `fog.svg`
- `wind.svg`
- `unknown.svg`

Recomendaciones:

- Formato SVG.
- Fondo transparente.
- `viewBox="0 0 64 64"`.
- Legibles sobre fondo oscuro `#071014`.
- Sin texto dentro del icono.
- Idealmente usar `currentColor` para poder colorearlos con CSS.
